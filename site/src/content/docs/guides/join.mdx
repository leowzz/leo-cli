## 选择输入

传入文件时，`leo join` 读取该 `.txt` 或 `.csv` 文件：

```bash
leo join ids.txt
leo join ids.csv
```

不传文件时，管道中的 stdin 优先于剪贴板：

```bash
seq 1 10 | leo join
leo join
```

文本以逗号或空白分隔。CSV 会尝试识别表头，并允许选择不同字段。

## 交互按键

- Left/Right 或 h/l：切换 CSV 字段。
- Up/Down：切换输出格式。
- u：切换唯一值和原始值；默认按首次出现顺序去重。
- Enter：接受结果并复制到剪贴板。
- Esc 或 Ctrl-C：取消。

## 输出格式

选择器提供四种格式：

```text
1,2,3
(1,2,3)
id in (1,2,3)
'1','2','3'
```

引号格式会把值中的单引号转义为两个单引号。确认后命令打印数量、预览和 `已复制到剪贴板`；取消时打印 `已取消`，且不会复制结果。
