---
title: 规范化剪贴板代码
description: 将剪贴板代码中的中文和全角符号转换为英文半角符号。
---

复制代码后运行：

```bash
leo norm
```

命令会读取整个剪贴板，转换后直接覆盖写回，并打印 `已规范化剪贴板`。例如：

```text
（a＝１，b＝“值”）  ->  (a=1,b="值")
```

转换范围包括全角 ASCII 字符和空格、中文逗号/句号/顿号、括号、方括号、书名号及中英文弯引号。其他字符保持不变。

`leo norm` 会处理剪贴板中的所有位置，包括字符串和注释。它不解析编程语言，也不会调整缩进或调用 gofmt、Prettier 等 formatter。
