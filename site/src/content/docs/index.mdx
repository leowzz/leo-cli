`leo-cli` 是一组面向个人开发工作流的命令行工具，涵盖仓库切换、SQL IN 构造、时间转换、镜像搬运和本地日志查看。

从源码构建：

```bash
make build
```

快速开始：

```bash
leo repo reindex
leo repo
eval "$(leo shell init zsh)"
```
